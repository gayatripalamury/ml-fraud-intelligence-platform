# ML-Powered Financial Fraud Intelligence Platform

An end-to-end Machine Learning platform designed to detect suspicious financial transactions, generate fraud risk scores, and visualize fraud insights through an interactive monitoring dashboard.

This system simulates a real-world **financial fraud monitoring platform** used by banks and fintech companies to analyze transactions and identify potential fraud patterns using Machine Learning.

---

## 🚀 Key Features

• Machine Learning based fraud detection using classification models  
• Transaction risk scoring based on behavioral patterns  
• Fraud explanation engine to identify suspicious triggers  
• Interactive monitoring dashboard built with Flask  
• Feature engineering for fraud pattern detection  
• Fraud analytics and insights visualization  
• Transaction submission and real-time fraud evaluation  
• SQLite database integration for transaction storage  

---

## 🧠 Machine Learning Pipeline

The system processes transactions through the following pipeline:
Transaction Input
↓
Feature Engineering
↓
Machine Learning Prediction
↓
Fraud Risk Score Generation
↓
Fraud Explanation Engine
↓
Database Storage
↓
Fraud Monitoring Dashboard

## 📊 Dashboard Capabilities

The dashboard provides real-time monitoring and analysis of transaction activity.

### 1. System Overview
Displays overall transaction statistics including:

- Total transactions processed
- High risk transactions
- Medium risk transactions
- Low risk transactions

### 2. Incoming Transactions
Displays the latest processed transactions along with their fraud risk scores.

### 3. Feature Analysis
Explains the behavioral indicators used to detect suspicious transactions.

Examples:
- Transaction amount deviation
- Night-time transactions
- Device change detection
- Location change detection

### 4. ML Model Decision
Displays fraud probability predictions generated by the trained ML model.

### 5. Fraud Explanation
Provides interpretable explanations for why a transaction may be suspicious.

### 6. Fraud Insights & Analytics
Shows fraud distribution and transaction risk patterns.

## ⚙️ Installation
Clone the repository:
git clone https://github.com/gayatripalamury/ml-fraud-intelligence-platform.git
Navigate into the project:
cd ml-fraud-intelligence-platform
Install dependencies:
pip install -r requirements.txt
---
## ▶️ Running the Application
Generate the dataset:
python data/generate_dataset.py
Train the Machine Learning model
python ml/train_model.py
Run the Flask application:
python app.py
Open the application in your browser:
http://127.0.0.1:5000
